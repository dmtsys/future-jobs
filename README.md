## Dev usage

``npm install --save future-jobs``

```javascript
import futureJob from 'future-jobs';
console.log(futureJob());
// unique one direction researcher
```

or you can pass a `seed` to generate a pseudo-random future job:

```javascript
futureJob("some_seed");
```
